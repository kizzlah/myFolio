# myFolio

A lightweight portfolio publishing platform built with plain HTML, CSS, and JavaScript.

## Features

- Search and filter portfolios by category
- Modal portfolio viewer
- Dark/light theme toggle
- Create and persist portfolios locally
- Responsive design
- No build system required

## Quick Start

1. Clone this repository
2. Serve it locally:
   ```bash
   # Using Python 3
   python3 -m http.server 8000
   ```
3. Open your browser to `http://localhost:8000`

## Development

This is a static site with no build process. Simply edit the files and refresh your browser.

- HTML entry point: `index.html`
- Styles: `src/styles/main.css`
- Scripts: `src/js/main.js`

## Adding Features

To add new portfolios or features:
1. Edit `index.html` to modify the markup
2. Update styles in `src/styles/main.css`
3. Add functionality in `src/js/main.js`

### Browser Compatibility

This site is tested in modern browsers that support:
- CSS custom properties
- CSS Grid/Flexbox
- ES6 JavaScript features
